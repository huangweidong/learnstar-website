# Gemini Nano Banana 文字卡片生图提示词

## 适用场景

小红书笔记封面图 — 教育类文字梗图/表情包卡片风格

## 基准提示词模板

```
生成一张小红书风格的竖版文字卡片，比例为 3:4。

背景为浅冰蓝色纯色，铺满整个画面，没有任何边框、描边或边界线。画面四周非常稀疏地点缀极少量手绘风格的装饰元素：两三颗粉色和黄色的小星星、两三个小蓝色爱心、一两条黄色的卷曲螺旋线条、几个粉色小圆点。装饰元素要极简克制，整体数量控制在十个以内，大面积留白，画面要透气干净。不要任何边框、外框或矩形描边。

主标题文字放在画面垂直方向的中上部位置，使用深藏青色的粗体手写毛笔字体，分两行居中排列：第一行写"我好像突然"，第二行写"就能管住纪律了"。其中"突然"两个字的背后有一个浅蓝色的矩形色块作为高亮强调。

主标题下方紧接着居中排列三个捂嘴偷笑的 emoji 表情。

整体画面风格清新、简洁、可爱，装饰元素仅起微微点缀作用，中央文字区域保持干净清爽。适合教育类小红书笔记使用。
```

## 可替换变量

| 变量 | 说明 | 示例 |
|------|------|------|
| 背景色 | 浅冰蓝色 → 其他浅色 | 浅粉色、浅绿色、浅黄色 |
| 主标题文案 | 分两行的标题文字 | 任意两行文案 |
| 高亮关键词 | 浅蓝色矩形色块高亮的词 | 替换为文案中的核心词 |
| 底部 emoji | 捂嘴偷笑 emoji | 替换为其他表情 |

## 调优经验

1. **装饰元素过多** → 加"极简克制"、"总数控制在十个以内"
2. **出现边框** → 明确写"没有任何边框、描边或边界线"、"不要任何边框、外框或矩形描边"
3. **四角太对称** → 加"分布自然随机、不对称，有的角落可以留空"
4. **蓝色笔刷条纹过多** → 去掉笔刷描述，或改为"最多出现一道淡蓝色短笔触"
5. **文字位置偏下** → 明确写"放在画面垂直方向的中上部位置"
