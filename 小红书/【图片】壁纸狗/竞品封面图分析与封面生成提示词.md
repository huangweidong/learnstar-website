# 小红书封面生成指南

> 使用 AI 生图工具（Gemini / Midjourney 等）批量生成小红书封面图。
> 每次生成时，上传产品截图作为参考图，配合以下提示词使用。

---

## 核心空间规则（所有提示词必须遵守）

> **屏幕是「背景墙」**：教室前方的大屏幕/电视占据画面背景的 70% 以上，像一面发光的墙壁。
> **人物站在屏幕正前方**：人在前景，屏幕在人的正后方，不是左右两侧。
> **构图关系**：人物遮挡屏��的一小部分（约 20-30%），屏幕内容在人物周围大面积可见。
>
> 错误示范：「人物在画面左侧 1/3，屏幕在右侧 2/3」— 这会让屏幕变成侧面墙上的东西。
> 正确示范：「人物站在大屏幕正前方，屏幕在身后充满整个背景，像在屏幕前拍自拍」。

## 四、场景类型库（基于竞品分析）

> 以下场景类型直接来自竞品图片的构图分析。

### A类：老师正面 + 屏幕背景（最高频）

```
- 老师站在教室前方大屏幕正前方，面向镜头
- 大屏幕在老师正后方，像一面墙壁充满整个背景
- 老师遮挡屏幕约 20-30%，屏幕内容在老师周围大面积清晰可见
- 老师可以在画面偏左或偏右，不一定居中
- 半身构图，从胸部或腰部以上入镜
```

### B类：老师 + 学生前景 + 屏幕背景（课堂感强）

```
- 镜头位于教室中后部，从学生座位区往前方拍
- 前景有 2-3 个学生的后脑勺/肩膀/侧面（虚化或清晰均可）
- 中景是老师站在屏幕前方，面朝学生方向或侧身
- 背景是大屏幕充满画面上方
- 有真实课堂的临场感和纵深感
```

### C类：老师背对镜头指屏幕（氛围感）

```
- 老师背对镜头，面朝屏幕
- 一只手抬起指向屏幕上的某个区域
- 头微微回头，露出好看的侧脸
- 屏幕在老师正前方（也就是画面背景），充满背景
- 大半身构图，背影线条优美
```

### D类：两位老师合影 + 屏幕背景

```
- 两位年轻女教师并排站在屏幕前方
- 一人在画面左侧，一人在右侧
- 其中一人可以指向屏幕，另一人面向镜头微笑
- 可以有差异化：一人戴眼镜/一人不戴，服装颜色不同
- 屏幕在两人正后方作为背景
- 近距离半身合影构图
```

### E类：学生视角 + 屏幕正前方（纯课堂感）

```
- 从教室后方往前方拍
- 前景是学生们的背影（后脑勺、肩膀、坐姿）
- 远处正前方是大屏幕，清晰显示软件界面
- 没有老师或老师只是侧面一小部分
- 完全的课堂实景感，像别人帮你拍的课堂照片
```

### G类：学生侧面触摸屏幕（互动感强）

```
- 学生站在屏幕旁边，侧面朝向屏幕
- 一只手伸出触摸屏幕上的具体内容（某张卡片、某个按钮）
- 学生在画面一侧（通常右侧），屏幕在另一侧，左右并列关系
- 从侧面拍，能看到学生的侧脸轮廓
- 表情专注或兴奋，像在挑选自己喜欢的东西
- 大半身构图
```

### H类：学生背影近距离触屏（代入感强）

```
- 非常近的特写，只看到学生的后脑勺和手
- 学生的手掌/手指按在屏幕上，触摸具体的卡片或按钮
- 屏幕充满整个背景，内容非常清晰
- 能看到手指触摸的具体位置
- 有强烈的第一视角代入感
- 可以看到学生的发型、眼镜、红领巾等小细节
```

### I类：学生群体涌向屏幕（热闹课堂）

```
- 从教室中后部拍向前方
- 5-6 个学生从座位上离开，围到教室前方的大屏幕前
- 有的学生伸手触摸屏幕，有的踮脚张望
- 屏幕在黑板中间位置，学生们挤在屏幕前方
- 前景有课桌、书本、还在座位上的学生
- 气氛热闹兴奋，像课间抢着看什么好东西
- 非常真实的课堂场景，生活感十足
```

### J类：两个学生一起看屏幕（同伴互动）

```
- 两个学生（可男可女，也可一男一女）站在屏幕前
- 侧面或背影入镜，一起看着屏幕
- 其中一个人手指指向屏幕上的某个内容
- 像在讨论"你看这个"的自然互动
- 屏幕充满背景，在两人前方或侧方
- 自然的同伴交流状态
```

### K类：手持手机展示APP + 电脑屏幕背景（多端展示）

```
- 一只手从画面右下方伸入，握持一部黑色手机
- 手机屏幕朝向镜头，显示软件的手机端界面
- 手机后方是一台普通黑色边框电脑显示器，显示同一款软件的电脑端界面
- 手机清晰，电脑屏幕自然虚化
- 生活化的书桌/办公桌场景
- 像随手拍的产品展示，不是广告摄影
```

### L类：老师侧身面向镜头 + 学生触屏互动（师生同框）

```
- 老师在画面右侧，面向镜头微笑
- 学生在画面左侧/中间，背影或侧面朝向屏幕，手指触摸屏幕
- 老师和学生同框，有温馨的师生互动感
- 大屏幕嵌在绿色推拉黑板中间，是真实的小学教室环境
- 半身构图，像同事帮拍的课堂瞬间
```

## 十、使用说明

### 生成流程

1. 从**第七章**选择要推的功能，截取对应产品界面
2. 从**排期表**选今天的组合，或自由选择场景类型+人物+服装
3. 复制对应的提示词（第八章），替换服装描述即可
4. 上传产品截图 + 提示词给 AI 生成
5. 选择最佳结果，用醒图/Canva 添加封面大字标题

### 注意事项

- 每次让 AI 生成 2 张供选择，挑最自然的一张
- 如果屏幕内容变形严重，可以让 AI 生成空白/模糊屏幕，后期手动贴截图
- 同一个人设连续发太多会被认出是 AI，建议每 2-3 天换一次服装和场景类型
- 封面大字不要让 AI 生成，AI 生成的中文字体不可控，用醒图/Canva 后期加
- 生成后检查：手指数量、文字是否乱码、光影是否自然
- **如果 AI 把屏幕放到侧面**：追加指令"屏幕应该在人物正后方作为背景墙，不是侧面"
- **A类场景最好用、出图质量最高**，建议作为主力构图
